About
=======

2016년 1차 천하제일도라대회때 사용했던 PHP 기반 소스코드 전체입니다.

급하게 만든 것이라 코드가 깔끔하진 않을지도 모릅니다.

현재는 이미 관련 서버가 사라진 상태입니다

* 주의: DB Query가 코드에 있는 이상한 코드입니다



프로젝트 정보
==============

수정된 사항: DB접속 정보

개발: Eclipse / LAMP+AWS


주요 기능
===========

관리자: 이용자 기록 등록 관리, 사용자 허가 및 취소

이용자: 대회 기록 등록

코-드: 대회 점수 자동 정렬
